# To-Do List Application

A modern, feature-rich to-do list application built with vanilla HTML, CSS, and JavaScript with local storage functionality.

## Features

✅ **Add Tasks** - Create new to-do items  
✅ **Mark Complete** - Check off finished tasks  
✅ **Edit Tasks** - Modify task description, priority, and due date  
✅ **Delete Tasks** - Remove individual tasks  
✅ **Filter Tasks** - View All, Active, or Completed tasks  
✅ **Local Storage** - Automatically saves and loads your tasks  
✅ **Statistics** - Shows total, active, and completed task counts  
✅ **Priority Levels** - Assign High, Medium, or Low priority  
✅ **Due Dates** - Add deadlines to tasks  
✅ **Responsive Design** - Works on desktop and mobile  
✅ **Beautiful UI** - Modern gradient design with smooth animations  

## How to Use

1. Open `index.html` in your web browser
2. Enter a task description in the input field
3. Click "Add Task" or press Enter
4. Use filter buttons to view different task categories
5. Click "Edit" to modify a task's description, priority, or due date
6. Click "Delete" to remove a task
7. Your data is automatically saved to local storage

- **CSS3** - Flexbox layout, gradients, animations
- **JavaScript (ES6)** - Class-based OOP design
- **Local Storage API** - Data persistence

## Local Storage

Tasks are automatically saved to the browser's local storage when you:
- Add a new task
- Mark a task as complete
- Edit a task
- Delete a task

Your data persists even after closing the browser!
